# Skills Catalog

Canonical list of skills available in `.kilocode/skills`.

## How to Use

- Trigger by name (example: `use skill create-pull-request`).
- Open the corresponding `SKILL.md`.
- Follow only the referenced files needed for the task.
- Use `registry.json` for automatic discovery by agents/tools.
