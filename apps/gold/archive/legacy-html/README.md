# Legacy HTML Archive

Este directorio conserva HTML históricos que ya no forman parte de la superficie activa ni del input MPA oficial.

## Propósito

- preservar historia del producto
- permitir referencia forense o rescate puntual
- evitar que superficies viejas sigan mezcladas con módulos vigentes

## Regla

Nada dentro de este árbol debe considerarse producto activo a menos que se reactive explícitamente en un lote nuevo.

Los HTML archivados se conservan como referencia histórica; no se garantiza que sigan ejecutándose de forma autónoma porque su bridge auth legacy también fue archivado fuera de la superficie activa.

- inventario operativo: `apps/gold/docs/LEGACY_SURFACES.md`
- fuente activa del repo: `apps/gold/docs/AGENT_REPORT_ACTIVE.md`
- histórico completo legacy: `apps/gold/docs/AGENT_REPORT.md`
