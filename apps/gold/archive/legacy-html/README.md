# Legacy HTML Archive

Este directorio conserva HTML históricos que ya no forman parte de la superficie activa ni del input MPA oficial.

## Propósito

- preservar historia del producto
- permitir referencia forense o rescate puntual
- evitar que superficies viejas sigan mezcladas con módulos vigentes

## Regla

Nada dentro de este árbol debe considerarse producto activo a menos que se reactive explícitamente en un lote nuevo.

## Relación con docs

- inventario operativo: `docs/LEGACY_SURFACES.md`
- estado corto del repo: `docs/AGENT_REPORT_ACTIVE.md`
- histórico completo: `docs/AGENT_REPORT.md`
