Preguntas frecuentes

Respuestas breves sobre el estado actual de YavlGold Agro V1, registro, datos y soporte.

Que es YavlGold?

YavlGold V1 es la release activa de una herramienta digital construida con Vanilla JS, Vite MPA y Supabase. Hoy funciona como Agro: control agricola y operativo para la finca real.

Como puedo registrarme?

Puedes crear una cuenta gratuita desde el flujo real de registro en la pagina principal. Usa el boton Comenzar Ahora o entra directamente a /#register.

Es gratis usar YavlGold?

Hoy, en YavlGold V1, el uso general de Agro es gratuito. No hay pagos ni suscripciones activas. Si se habilitan planes o cobros, se informara antes de aplicarlos.

Puedo usar Agro sin internet?

Agro funciona principalmente online porque guarda datos con Supabase, consulta clima y sincroniza informacion de tu cuenta. Algunas superficies puntuales, como AgroRepo, usan LocalStorage en ese contexto especifico.

Que modulos estan disponibles hoy?

La experiencia activa del producto es Agro. Al entrar a YavlGold entras a una herramienta para registrar cultivos, movimientos, tareas y seguimiento real del campo.

Como contacto a soporte?

Para consultas de uso, errores o solicitudes de privacidad puedes escribir a soporte@yavlgold.com.

Que significa V1?

V1 identifica el release visible activo del producto. Las referencias a V9.8 quedan solo como contexto historico cuando un documento o cronica lo marque expresamente.
